Remapping controls

To remap a control, go into the Control Panel and make sure that it has connected, if not see Troubleshooting.

Then move the control on the midi controller, this should cause the control's data to be filled out in the "Last Moved Control" table as follows:

Name Value Device Output Inverted Channel Ch Device Ch Output Action Action Output
The name of the control The value of the control The output device to which the control has been mapped The path to which the control has been mapped Whether the control is inverted The channel group of which the control is a part The output device to which the channel group has been mapped The type of the channel to which the channel group has been mapped The action group of which the control is a part The type of the action to which the action group has been mapped
Always filled Only filled if control individually mapped Always filled Only filled if channel mapped Always filled Only filled if action mapped

After selecting the desired control click "Enter Edit Mode" to begin editing, then you can set the mapping for the control, the channel or the action as described in Introduction
