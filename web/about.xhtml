About midi2osc2

midi2osc2 runs as a server component on a device, such as a raspberry pi, connected to your midi controller and can be configured using any web browser using a websocket driver interface in realtime. The web interface does not have to be served from the same device that is running the server, however the client must be able to open a connection on port 8080 to the server.
