Config Files

midi2osc2 is configured by a main config file that is passed as the first and only command line argument. This references several other files also described in this document.

The main config file

The main config file has should contain only lines in the following formats and comments (lines starting with //). Leading whitespace is ignored.
Format Count Description
midi:<Device name>:<Profile Path> Exactly once The name and profile of the midi controller. The name must be a prefix of how the midi controller appears on the system, this is to allow differences between operating systems. The profile path must be a path to a file as described below
