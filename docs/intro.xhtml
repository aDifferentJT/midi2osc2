Introduction

midi2osc2 is a program for mapping controls on a Midi controller to OSC paths. It also supports bank switching and multiple outputs.

Each control on the midi controller is part of both a channel group and an action group. Channel groups can be mapped to various types of channel and action groups can be mapped to various types of actions. As well as this individual controls can be mapped to OSC paths and potentially inverted.

When a control is moved on the midi controller then if the control has an individual mapping then that is used, otherwise, if the control's channel group and action's group both have mappings then they are combined and the resulting output path used, otherwise the control is ignored.
